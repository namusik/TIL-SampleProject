## Helm values 전체 조회
```sh
helm get values <ARGOCD_RELEASE> -n <ARGOCD_NS> -a
```
여기서 확인해야 할 핵심 포인트
- configs
  - argocd-cm, argocd-rbac-cm로 내려간 설정의 “원본”
- dex 또는 server.config / oidc.config
  - SSO 구조 파악
- server.ingress
  - ALB / 인증서 / 외부 노출 방식
- repositories, credentialTemplates
  - Bitbucket 연동 방식

- argo cd 에서 관리하는 application 보기
